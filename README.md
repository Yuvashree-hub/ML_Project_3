# Resume Skill Extractor

## Overview
This project builds a simple Resume Skill Extractor using Python. The system reads resume text and automatically identifies important technical skills from it.

## Features
- Extracts skills from resume text
- Uses text processing techniques
- Simple and easy to use
- Works with resume datasets

## Technologies Used
- Python
- Pandas
- Regular Expressions (re)

## Dataset
Dataset used: Resume Dataset

File used:
Resume.csv

## How It Works
1. The system reads the resume dataset.
2. Resume text is converted to lowercase.
3. A predefined list of technical skills is used.
4. Regular expressions search for these skills in the resume text.
5. Extracted skills are displayed as output.
